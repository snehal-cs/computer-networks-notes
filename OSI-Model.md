# OSI Model

1. Physical Layer
- Responsible for transmitting raw bits.
- Devices: Hub, Cable.

2. Data Link Layer
- Responsible for data transfer between devices.
- Devices: Switch.
- Protocol: Ethernet.

3. Network Layer
- Responsible for logical addressing and routing.
- Protocol: IP.
- Device: Router.

4. Transport Layer
- Responsible for end-to-end communication.
- Protocols: TCP, UDP.

5. Session Layer
- Establishes and manages sessions.

6. Presentation Layer
- Handles data formatting and encryption.

7. Application Layer
- Provides network services to applications.
- Protocols: HTTP, HTTPS, DNS, FTP.
